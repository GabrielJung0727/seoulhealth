========================================

1. 관리자용 안내
========================================

[관리자 로그인 방법]

1. 주소창에 <https://doublej.app/login> 입력
2. 관리자 비밀번호 입력 후 로그인 버튼 클릭
3. 관리자 대시보드가 열립니다

관리자 대시보드는 한국어로 되어 있으며, 글씨와 버튼이 크게 되어 있습니다.

[관리자 대시보드 사용법]

대시보드에 들어가면 상단에 전체 지원서 수와 문의 수가 큰 숫자로 표시됩니다.

1) 지원서 확인하기
   - "지원서" 탭을 누르면 전문가 네트워크에 지원한 분들의 목록이 나옵니다
   - 카드를 누르면 상세 정보를 볼 수 있습니다 (이름, 이메일, 경력, 학력, 전공 등)

2) 문의 확인하기
   - "문의" 탭을 누르면 서비스 문의 목록이 나옵니다
   - 카드를 누르면 문의 제목, 내용, 회사정보 등을 볼 수 있습니다

3) 상태 변경하기
   - 카드를 눌러 상세 화면을 열면, 하단에 큰 버튼 3개가 있습니다
   - 신규 / 검토완료 / 연락완료 중 원하는 상태를 누르면 됩니다
   - "정말 변경하시겠습니까?" 확인 팝업이 뜨면 "확인"을 누르세요

4) 검색하기
   - 상단 검색창에 이름이나 이메일을 입력하면 해당 건만 보여줍니다

5) 상태별 필터
   - "전체 / 신규 / 검토완료 / 연락완료" 버튼을 눌러 해당 상태만 볼 수 있습니다

6) 엑셀 다운로드
   - "CSV 내보내기" 버튼을 누르면 목록이 엑셀에서 열 수 있는 파일로 다운로드됩니다

7) 인쇄
   - 상세 화면에서 "인쇄" 버튼을 누르면 해당 건을 인쇄할 수 있습니다

8) Q&A 답변
   - 회사 고객이 Q&A 게시판에 질문을 남기면 관리자 대시보드에서 확인하고 답변할 수 있습니다
   - 답변을 작성하면 해당 회사 이메일로 알림이 자동으로 갑니다

[문의가 들어오면 어떻게 되나요?]

1. 해외 고객이 사이트에서 문의 폼을 작성하고 제출합니다
2. 관리자 임시 이메일(<doublej.biz01@gmail.com>)로 알림 메일이 자동으로 옵니다
3. 고객에게도 "접수되었습니다" 확인 이메일이 자동으로 갑니다
4. 관리자 대시보드에 접속하면 해당 문의를 확인할 수 있습니다
5. 상태를 "검토완료"나 "연락완료"로 변경하시면 됩니다

[전문가 지원이 들어오면 어떻게 되나요?]

1. 전문가가 사이트의 "Our Expert Network" 페이지에서 지원서를 제출합니다
2. 관리자 이메일로 알림이 옵니다
3. 대시보드 "지원서" 탭에서 확인할 수 있습니다
4. 경력, 학력, 전공 등을 검토한 후 상태를 변경하시면 됩니다

========================================
2. 고객 회사용 안내
========================================

[고객 회사가 볼 수 있는 화면]

고객 회사는 사이트에서 아래 페이지들을 이용할 수 있습니다.

1) 메인화면 (<https://doublej.app>)
   - 회사 소개, 4개 전문 분야, 서비스 안내

2) About Us (<https://doublej.app/about>)
   - 대표 인사말, 대표 경력, 추천글

3) Our Expert Network (<https://doublej.app/network>)
   - 전문가 네트워크 소개
   - 하단 "APPLICATION FORM" 버튼 -> 전문가 지원 양식 팝업

4) Our Process (<https://doublej.app/process>)
   - 문의 접수부터 보고서 전달까지 7단계 프로세스 안내

5) Request Now (<https://doublej.app/request>)
   - "INQUIRY FORM" 버튼 -> 서비스 문의 양식 팝업

6) 회원가입 (<https://doublej.app/company/register>)
   - 회사명, 담당자명, 이메일, 비밀번호, 전화번호, 국가, 산업분야 입력

7) 로그인 (<https://doublej.app/company/login>)
   - 이메일 + 비밀번호 입력 후 이메일로 받은 6자리 인증코드 입력

8) 회사 대시보드 (로그인 후 <https://doublej.app/company/dashboard>)
   - Overview: 내 문의 건수, Q&A 건수 확인
   - My Inquiries: 내가 보낸 문의 목록과 처리 상태 확인
   - Settings: 회사정보 수정, 비밀번호 변경

9) Q&A 비밀게시판 (로그인 후 <https://doublej.app/company/qa>)
   - 관리자에게 비밀 질문 작성
   - 채팅 형태로 질문/답변 주고받기
   - 다른 회사의 Q&A는 볼 수 없음 (비공개)

[회원가입 방법 2가지]

방법 1) 직접 회원가입

- 사이트 우측 상단 "LOGIN" 클릭 -> 하단 "Register" 클릭
- 회사명, 담당자명, 이메일, 비밀번호(8자 이상), 전화번호, 국가, 산업분야 입력
- 가입 완료 시 환영 이메일 발송

방법 2) 폼 작성 시 자동 가입

- 회원가입 없이 "Request Now"나 "Application Form"에서 폼을 작성해서 제출
- 입력한 이메일로 계정이 자동 생성됨
- 임시 비밀번호가 이메일로 발송됨
- 해당 이메일과 임시 비밀번호로 로그인 후 비밀번호 변경 가능

[로그인 후 폼 작성 시 자동 입력되는 항목]

로그인한 상태에서 문의 폼(INQUIRY FORM)이나 지원 폼(APPLICATION FORM)을 열면
아래 항목이 자동으로 채워집니다. 직접 입력할 필요가 없습니다.

- 이름 -> 회원가입 시 입력한 담당자명
- 이메일 -> 회원가입 시 입력한 이메일
- 전화번호 -> 회원가입 시 입력한 전화번호 + 국가코드
- 현 직장 (문의 폼만) -> 회원가입 시 입력한 회사명

나머지 항목(경력, 문의 내용 등)은 직접 작성해야 합니다.

[로그인 보안 (2단계 인증)]

고객 로그인은 2단계 인증으로 보안이 강화되어 있습니다.

1단계: 이메일 + 비밀번호 입력
2단계: 이메일로 6자리 인증코드(OTP)가 발송됨 -> 10분 안에 입력

비밀번호를 잊어버린 경우:

- 로그인 화면에서 "Forgot Password?" 클릭
- 가입한 이메일 입력
- 임시 비밀번호가 이메일로 발송됨
- 로그인 후 Settings에서 비밀번호 변경

3. 자동 이메일 시스템
========================================

아래 상황에서 이메일이 자동으로 발송됩니다.

[관리자에게 가는 이메일]

- 고객이 문의 폼(Inquiry Form)을 제출했을 때 -> 문의 내용 알림
- 전문가가 지원서(Application Form)를 제출했을 때 -> 지원 내용 알림
- 고객이 Q&A 게시판에 새 질문을 작성했을 때 -> 질문 알림

[고객에게 가는 이메일]

- 문의 폼 제출 후 -> "접수되었습니다" 확인 이메일
- 지원서 제출 후 -> "접수되었습니다" 확인 이메일
- 비회원이 폼 제출 시 -> 자동 계정 생성 + 임시 비밀번호 안내 이메일
- 회원가입 시 -> 환영 이메일
- 로그인 시 -> 6자리 인증코드(OTP) 이메일
- 비밀번호 찾기 시 -> 임시 비밀번호 이메일
- 관리자가 Q&A에 답변했을 때 -> 답변 알림 이메일

총 10가지 종류의 이메일이 자동 발송됩니다.
